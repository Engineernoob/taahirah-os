🎧 NotchPod

NotchPod is a desktop-first audio companion that showcases a tactile playback interface inspired by digital jog wheels. It couples rich motion design with precise audio control to deliver a focused, immersive listening experience.

✨ Features

  • 🎚️ Wheel-driven transport with play, pause, skip, and scrubbing gestures
  • 🎨 Theme cycling via dynamic accent colors from track artwork
  • 🔊 Smooth audio playback powered by Howler.js
  • ⌨️ Keyboard shortcuts for global playback control
  • 🪩 Animated glassmorphism UI with Framer Motion
  • 🧠 State management with Zustand + persistent themes
  • 🖥️ Tauri integration for lightweight desktop builds

⚙️ Tech Stack

  • React 19 + TypeScript
  • Vite for hot reloading and bundling
  • Framer Motion for animations
  • Howler.js for audio control
  • Zustand for state management
  • Tauri for cross-platform desktop packaging

🚀 Getting Started

# install dependencies
npm install

# start the web development server
npm run dev

# build the web bundle
npm run build

# preview the production build locally
npm run preview

# run the Tauri desktop app
npm run tauri:dev

# produce a desktop build
npm run tauri:build

📂 Project Structure

  • src/ – React components, hooks, and UI logic
  • public/ – static assets (cover art, icons)
  • src-tauri/ – Tauri configuration & Rust backend
